# Data-Science-Lab-Work
House Price Prediction Using Machine Learning

Description

The objective of this project is to predict house prices based on housing features using a Linear Regression model. This project demonstrates the application of regression techniques for real-world price prediction problems.

Dataset

- Source: Scikit-learn California Housing Dataset
- Description of Data:
  The dataset contains housing information from California districts, including:
  - Median Income
  - House Age
  - Average Rooms
  - Average Bedrooms
  - Population
  - Average Occupancy
  - Latitude
  - Longitude

Steps Performed

1. Data Loading and Inspection
2. Data Cleaning (Null/Duplicate Check)
3. Exploratory Data Analysis
4. Data Visualization
5. Feature Engineering
6. Model Building using Linear Regression
7. Prediction and Evaluation

Results

- Key Findings:
  
  - Median income and location were the most influential features affecting house prices.
  - The model was able to predict house prices with reasonable accuracy.
  - Predicted values were generally close to actual values for many samples.

- Metrics:
  
  - Mean Squared Error (MSE): 0.5559
  - Root Mean Squared Error (RMSE): 0.7456
  - Mean Absolute Error (MAE): 0.5332
  - R² Score: 0.5758

Tools Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

Conclusion

The Linear Regression model was able to predict house prices with reasonable accuracy. Median income and location were found to be the most influential features. This project demonstrates the practical use of regression techniques for real-world house price prediction tasks.
